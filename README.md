# 分布式原理

*互联网发展至今，为了应对更高的业务量、提供更优质的服务，后端从单体服务逐渐演进至当下主流的分布式架构。所谓分布式，通俗来说就是将一个大而全的服务，基于业务领域拆分为多个独立的子服务，并针对每个子服务部署合适的实例，不通的实例在空间上是可以随意分布的。
